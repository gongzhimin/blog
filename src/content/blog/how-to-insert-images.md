# 如何插入图片

在这篇教程中，我会演示两种在博客文章中插入图片的方法。

## 方法一：使用 public 文件夹（推荐）

### 步骤 1：放置图片

将你的图片放到 `public/images/` 目录下。例如：

```
blog/
├── public/
│   └── images/
│       └── my-photo.jpg
```

### 步骤 2：在文章中引用

在 Markdown 文章中使用以下语法：

```markdown
![图片描述](/images/my-photo.jpg)
```

![图片示例](/images/image-guide.svg)

---

## 方法二：使用外部链接

如果图片托管在其他地方（如图床），可以直接使用 URL：

```markdown
![图片描述](https://example.com/image.jpg)
```

---

## 最佳实践

1. **图片压缩** - 上传前先压缩图片，减少加载时间
2. **合理命名** - 使用有意义的文件名，如 `2026-04-beach-sunset.jpg`
3. **Alt 文本** - 记得填写图片描述，这对可访问性和 SEO 都很重要
4. **图片大小** - 建议单张图片不超过 1MB

---

## 写在最后

插入图片非常简单，只需要记住 `/images/文件名` 这个路径格式即可。
